# Design Overview

## Objective
The objective of this project was to design a compact control board that could support monitoring and thermal management functions in a small edge or self-hosted computing environment.

## Functional Scope
The board provides:
- regulated power rails
- embedded monitoring and control
- temperature sensing
- cooling fan control
- visual status indication
- debug connectivity

## Design Philosophy
The design prioritized reliability, clarity of layout, modularity, and documentation quality. It was approached as if it were an early-stage product prototype rather than a one-off circuit exercise.

## Constraints
- Compact footprint
- Stable power conversion from a 12V input source
- Separation of noisy power circuitry from sensitive logic areas
- Clear routing for debug and communication lines
- Practical component availability
