# Bring-Up Checklist

## Pre-Power Checks
- inspect component placement and polarity
- inspect solder joints
- confirm no visible bridges
- verify continuity on major power nets
- verify no short between power and ground

## Initial Power-On
- use current-limited bench supply
- apply 12V input
- observe current draw
- verify no unexpected heating
- verify power LED behavior

## Rail Verification
- measure 5V rail
- measure 3.3V rail
- compare to expected tolerances

## Interface Verification
- verify microcontroller startup
- verify UART output
- verify I2C temperature sensor response
- verify PWM fan behavior
- verify fault indicator logic
